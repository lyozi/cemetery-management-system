# Full Stack Cemetery Management System with Interactive Maps

**Thesis Project (BSc)** - Budapest University of Technology and Economics

## Overview
A web application for cemetery registry and virtual memorialization. It features an interactive map based on drone imagery, allowing users to locate graves and view memorial pages.

## Tech Stack
* **Frontend:** React, TypeScript, Chakra UI, Leaflet
* **Backend:** ASP.NET Core (.NET 8), Entity Framework Core
* **Database:** PostgreSQL
* **Architecture:** Clean Architecture (Onion)

## Key Features
* Interactive Map with custom tile generation from drone footage.
* Virtual memorial pages (lighting candles, leaving messages).
* Admin dashboard for managing grave polygons and deceased records.
